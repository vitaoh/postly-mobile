# Postly

O **Postly** é uma rede social mobile focada em compartilhamento rápido de conteúdo e interação entre usuários.

### Principais funcionalidades

* 📱 **Feed infinito** com as abas **For You** e **Seguindo**.
* ❤️ Possibilidade de **curtir** e 💬 **comentar** publicações.
* ✉️ **Chat em tempo real** entre usuários.
* 🖼️ Envio de **imagens** nas conversas.
* 🎤 Suporte a **mensagens de voz**.
* 📸 Recurso de acesso rápido à câmera: ao **balançar o celular**, o aplicativo abre a câmera para capturar uma foto e criar uma nova publicação.

O projeto está sendo desenvolvido para oferecer uma experiência moderna, dinâmica e intuitiva, conectando pessoas por meio de conteúdo e comunicação instantânea.
